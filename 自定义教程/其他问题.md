# 其他问题

## 删除原版配方

在任意自定义JSON文件中添加。

```json
{
  "remove_recipe": "minecraft:mushroom_stew", // 配方ID
  // 或
  "remove_recipes": [ "minecraft:mushroom_stew" ] // 配方ID
}
```

## 调整堆叠上限

在食物JSON文件中添加。

```json
{
  "stack_size": 64
}
```

## 删除容器

在食物JSON文件中添加。

```json
{
  "remove_container": true
}
```
